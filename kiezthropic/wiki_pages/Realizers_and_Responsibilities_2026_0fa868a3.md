# **❗** Default World Office

While Production deals with the material structure, the Default World Office sorts out the invisible framework we need to exist legally as an event. They deal in advance with the Radical Bureaucracy of the Default World, so we can forget it exists during the event. A lot of these roles are pretty quick, but they need to be done or we can't burn! Good German skills and an appreciation for paperwork are desirable qualities in this circle.
