## Meal plan

Coordination Sheet: <https://docs.google.com/spreadsheets/d/1k0JHDJiw5JkRTOQm6CeFvtT1HG8lkdLJqSs6JsO56sA/edit?gid=586777720#gid=586777720>

For every day there is one tab to collect the meals for that day. I've filled out Friday 19th and Saturday 20th as examples. Obviously we can change everything We fill out the Ingredient and Amount For 1 columns, and then the amount scales automatically. The Measurement column is there to make sure we don't use "grams of beans" in one meal and "can of beans" in another, which would make consolidation for the shopping list really hard.

Instead every ingredient is listed in the Ingredient Tab. There the measurement is also set (which we can change as well, but should do so only once at the start). This is the list our camp had last year, with price estimates based on Rewe Online (also from last year so not exactly up to date).

These price estimates are then used to calculate the total price, and price per meal for everything. You can see what I've put in so far in the Price per Meal tab. Also from the Ingredients tab, we know which ingredients we can order a bit earlier and store vs which we should buy as late as possible. This is then all combined in the Non Perishable Shopping List and the Perishable Shopping List.

Finally there is a tab for General Stuff. It has the same columns as the daily tabs, to make it possible to combine them.

If anything breaks, just let Rene know. It is a bit more complicated, but in my experience this is worth it by reducing errors later.
